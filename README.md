# How to run (quick start)

The project uses **SQLite** by default, so you can run it without installing MySQL.

## 1. Open a terminal in the project folder

```bash
cd django-crud-ajax-login-register-fileupload
```

## 2. Create a virtual environment with uv (do this first)

Install [uv](https://docs.astral.sh/uv/) if you haven’t (e.g. `pip install uv` or see [install guide](https://docs.astral.sh/uv/getting-started/installation/)).

Create the virtual environment:

```bash
uv venv
```

Activate it:

**Windows (PowerShell):**
```powershell
.\.venv\Scripts\Activate.ps1
```

**Windows (Command Prompt):**
```cmd
.venv\Scripts\activate.bat
```

**Linux / macOS:**
```bash
source .venv/bin/activate
```

## 3. Install dependencies

With the virtual environment activated:

```bash
uv pip install -r requirements.txt
```

## 4. Run migrations (create the database)

```bash
python manage.py migrate
```

## 5. Create a superuser (admin account)

```bash
python manage.py createsuperuser
```

Enter username, email, and password when prompted.

## 6. Start the development server

```bash
python manage.py runserver
```

## 7. Open the app

- In your browser go to: **http://127.0.0.1:8000**
- Log in with the superuser account you created, or register a new user at **http://127.0.0.1:8000/register/**

---

# Optional: use MySQL instead of SQLite

1. Install MySQL and create a database.
2. Install the MySQL client: `pip install mysqlclient`
3. Set environment variables before running the app:
   - `USE_MYSQL=1`
   - `DB_NAME=your_db_name`
   - `DB_USER=your_db_user`
   - `DB_PASSWORD=your_db_password`
   - `DB_HOST=localhost` (optional, default is localhost)
   - `DB_PORT=3306` (optional)

Then run `python manage.py migrate` and `python manage.py runserver` as above.
